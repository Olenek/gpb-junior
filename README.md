## GPB Junior Task

Решением являются 3 jupyter ноутбука:
- `train_ols.ipynb` - тренировка OLS модели по численным параметрам данных на 80% семпла.
- `train_bert.ipynb` - тренировка BERT модели для предсказания ошибок OLS модели на том же подмножестве семпла.
- `eval.ipynb` - оценка и интерпретация решения

Соответственно стоит запускать их именно в таком порядке, запуск требует поместить исходные данные в директорию `data/`.

В директории `src/` хранится имплементация импьютера и враппера над `statsmodels` для привода OLS к `sklearn` формату.

Используемые пакеты:
- `statsmodels`
- `sklearn`
- `pytorch`
- `transformers`
- `datasets`
- `numpy`
- `pandas`
